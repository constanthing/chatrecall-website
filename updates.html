What's new in ChatsRecall

0.14.0 January 14, 2026

Added

  • Added "through your chats" text in header beside "ChatsRecall" text
  • Down arrow in "Update Library" page to indicate user can scroll down more (auto-scrolls when clicked)
  • Guide to "Update Library" page - runs immediately for first time users, with guide button to reactivate
  • Additional instruction to export steps: "Drop the conversations.json file below"
  • Settings page with Hotkeys, Appearance, and Support sections
  • Keyboard navigation for chats (Arrow Up/Down to navigate, Enter to open, Esc to deselect)
  • Title search highlighting (previously only prompts were highlighted)

Changed

  • Accent colors for Wave and Perplexity (previously too similar to each other)
  • Default theme now matches user's system theme

Fixed

  • Critical bug where new prompts were not being dynamically added in ChatGPT conversations
  • Title renaming bug for Claude chats on occasional failures to load
  • Claude chats retitling to include "- Claude" by removing suffix from titles
  • Scraper infinite loop edge case in scroll while loop

Removed

  • Theme button (replaced with settings button; theme moved to settings page)
  • Context menu (plan to gauge real user needs once released with in-popup surveys)

Removed
